# ToDo アプリ Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Rails 8.1 + Hotwire (Turbo Frames) で動作するシンプルな ToDo アプリを構築する。

**Architecture:** Todo モデル（title/completed）+ TodosController（index/create/update/destroy）。ビューは Turbo Frames を使い、ページリロードなしでタスクの追加・完了トグル・削除を実現する。Turbo Stream レスポンスでリストを部分更新する。

**Tech Stack:** Ruby on Rails 8.1, SQLite3, Hotwire (Turbo), 素の CSS

- [ ] **Step 4: コントローラを実装する**

`app/controllers/todos_controller.rb` を作成する:

```ruby
class TodosController < ApplicationController
  def index
    @todos = Todo.order(created_at: :desc)
    @todo = Todo.new
  end

  def create
    @todo = Todo.new(todo_params)
    if @todo.save
      @todos = Todo.order(created_at: :desc)
      respond_to do |format|
        format.turbo_stream
        format.html { redirect_to todos_path }
      end
    else
      respond_to do |format|
        format.turbo_stream { render turbo_stream: turbo_stream.replace("todo_form", partial: "form", locals: { todo: @todo }) }
        format.html { render :index, status: :unprocessable_entity }
      end
    end
  end

  def update
    @todo = Todo.find(params[:id])
    @todo.update!(todo_params)
    respond_to do |format|
      format.turbo_stream
      format.html { redirect_to todos_path }
    end
  end

  def destroy
    @todo = Todo.find(params[:id])
    @todo.destroy!
    respond_to do |format|
      format.turbo_stream
      format.html { redirect_to todos_path }
    end
  end

  private

  def todo_params
    params.require(:todo).permit(:title, :completed)
  end
end
```

`app/views/todos/_todo.html.erb` を作成する:

```erb
<%= turbo_frame_tag dom_id(todo) do %>
  <div class="todo-item <%= todo.completed ? 'completed' : '' %>">
    <%= form_with model: todo do |f| %>
      <%= f.check_box :completed, onchange: "this.form.requestSubmit()", checked: todo.completed %>
    <% end %>
    <span class="todo-title"><%= todo.title %></span>
    <%= button_to "削除", todo_path(todo), method: :delete, class: "delete-btn", form: { data: { turbo_confirm: "削除しますか？" } } %>
  </div>
<% end %>
```
